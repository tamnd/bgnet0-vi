## Guidelines

- Translate meaning, not words; prefer natural Vietnamese phrasing.
- Keep all code blocks, variable names, function names, and protocol names
  in English.
- Technical terms: English first, Vietnamese gloss in parens on first
  occurrence per chapter.
- No em-dashes (`—`); use `---` (three hyphens) as in the source.
- Preserve all markup tags verbatim: `[i[...]i]`, `[flw[...]]`, etc.
- No AI-boilerplate headers in translation text.
